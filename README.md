## PC端

### uart通信
这个Uart类是一个串口通信的封装类，提供了UART设备的操作接口。

| 核心方法       | 功能描述           |参数说明                                  | 
| ------------- |------------------|-----------------------------------------|
| open()        | 打开指定的串口设备  |port: 串口名称；baud_rate: 波特率，默认1Mbps |
| close()       | 关闭串口连接       |                                         |
| tx()          | 向串口发送字节数据  |bytes：数据类型                            |
| rx()          | 从串口读取指定字节数|size：读取的字节数，默认为1                  |



## FPGA端

### 通信

#### uart

| 核心模块       | 功能描述           |参数说明                                  | 
| ------------- |------------------|-----------------------------------------|
|  uart_tx()    | 异步串行通信（UART）的发送器模块，用于将并行数据转换为串行数据发送出去。  |CLK_COUNT：波特率分频系数，每个位持续868个时钟周期868；BIT_COUNT：固定为8位数据位 |
|  uart_rx()    | 异步串行通信（UART）的接收器模块，用于将串行数据转换为并行数据接收进来。  |CLK_COUNT：波特率分频系数，每个位持续868个时钟周期868；BIT_COUNT：固定为8位数据位 |

uart_tx()运行逻辑  
1.数据储存：定义[8:0]数据寄存器。  
2.运行时序：   <1>IDLE → START_BIT：当i_write_en有效时进入发送状态。  <2>START_BIT → DATA_BIT：发送完起始位后（经过CLK_COUNT个时钟）。  <3>DATA_BIT → STOP_BIT：发送完8个数据位后。  <4>STOP_BIT → IDLE：发送完停止位后。  
3.数据输出： 开始输出时，o_tx信号置0，o_tx信号串行输出储存的八位数据，当停止位和空闲时o_tx为1。
